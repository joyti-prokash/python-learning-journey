Medical Note System Project

A simple command-line medical note management system built in Python.

This project began as a text-file note saver and gradually evolved into a modular application with support for viewing, editing, deleting, and organizing patient notes.

Features
Add patient notes
View saved notes
Edit existing notes
Delete notes

medical_note_system_project/

main.py → application menu

notes.py → note management functions

patient_notes.txt → text storage
