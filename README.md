# Python Learning Journey

This repository documents my progression in learning Python through small exercises, file handling, CSV processing, and project development.

The goal of this repository is not to present polished production software, but to show how I gradually built programming skills and applied them in increasingly structured projects.

## Repository Structure

### fundamentals/

Basic Python concepts and programming exercises.

Topics:

* Variables and input/output
* Conditions and loops
* Functions
* Lists and strings
* Simple console applications

Example files:

* calculator.py
* for_loop.py
* simple_login_check.py
* student_average.py

### files-and-csv/

Projects focused on working with files and structured data.

Topics:

* Reading and writing text files
* CSV data processing
* Organizing records
* Simple data reporting

Example files:

### medical-notes-evolution/

An iterative project series showing how a medical note application evolved.

Topics:

* Saving patient notes
* Viewing records
* Editing and deleting entries
* CSV storage
* Auto-generated IDs
* Modular design

Example files:

* patient_note_saver.py
* patient_note_reader.py
* medical_note_system_v5.py
* add_note_with_id.py
